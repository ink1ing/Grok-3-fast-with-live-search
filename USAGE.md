# Grok 3.0 Chat 应用使用说明

## 🚀 功能特性

### ✨ 全新界面设计
- **现代化 UI**: 采用 GitHub Dark 主题风格，支持深色模式
- **响应式设计**: 完美适配桌面、平板和移动设备
- **流畅动画**: 消息滑入动画、按钮悬停效果等
- **智能布局**: 自适应侧边栏和聊天区域

### 📝 Markdown 渲染支持
- **完整 Markdown 语法**: 支持标题、列表、链接、图片等
- **代码高亮**: 使用 highlight.js 提供语法高亮
- **数学公式**: 支持 LaTeX 数学公式渲染 (MathJax)
- **表格支持**: 完美渲染 Markdown 表格
- **引用块**: 支持多级引用和特殊样式

### 🔍 xAI Live Search 集成
- **实时搜索**: 获取最新的网络信息
- **多数据源**: 支持 X、网页、新闻、RSS 等
- **智能决策**: AI 自动判断是否需要搜索
- **精准回答**: 基于实时数据提供准确回复

### 💬 智能对话管理
- **多轮对话**: 支持上下文连续对话
- **会话历史**: 自动保存和管理对话记录
- **快速切换**: 一键切换不同对话
- **会话删除**: 支持删除不需要的对话

## 🎯 使用方法

### 1. 初始设置
1. 打开浏览器访问 `http://localhost:10000`
2. 点击左下角的"设置"按钮
3. 输入您的 xAI API 密钥 (格式: `xai-xxxxxxxxxx`)
4. 选择是否启用 Live Search 功能
5. 点击"保存"完成设置

### 2. 开始对话
1. 在底部输入框中输入您的问题
2. 按 Enter 键或点击"发送"按钮
3. 等待 Grok 3.0 的回复
4. 支持 Shift+Enter 换行输入

## 🚨 故障排除

### 常见问题

#### 1. API 密钥无效
- 确保密钥格式正确: `xai-[50位字符]`
- 检查密钥是否有效且未过期
- 确认账户有足够的使用额度

#### 2. 连接问题
- 检查网络连接
- 确认防火墙设置
- 尝试刷新页面重新连接

#### 3. Markdown 渲染问题
- 确保使用标准 Markdown 语法
- 代码块需要使用三个反引号包围
- 数学公式使用 $ 或 $$ 包围

#### 4. 移动端显示问题
- 确保使用现代浏览器
- 检查是否启用了 JavaScript
- 尝试横屏/竖屏切换

### 性能优化建议
1. **定期清理**: 删除不需要的对话历史
2. **网络优化**: 使用稳定的网络连接
3. **浏览器更新**: 保持浏览器为最新版本
4. **缓存清理**: 定期清理浏览器缓存
